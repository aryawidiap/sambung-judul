# Sambung Judul: The Song Title Connecting Game

Welcome to Sambung Judul! This is a personal project I created inspired by a game my friends and I played while travelling around.

![Sambung Judul at a glance](./public/sambung-lagu-overview.png)

Visit the project website and play the game [here](https://aryawidiap.github.io/sambung-judul/).

## Tools Used

- Next.js
- React
- Tailwind CSS
- Motion (JavaScript and React animation library)
- `stopwords` npm package
- `array-of-english-words` npm package
